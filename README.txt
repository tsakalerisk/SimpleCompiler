Υλοποίηση ενός απλού compiler, σε jvm.

Για να δημιουργήσετε τον compiler : make
Για να διαγράψετε τα αρχεία του compiler: make clean
Για να διαγράψετε τα μεταγλωττισμένα αρχεία των παραδειγμάτων: make cleanFiles.
Στον φάκελο Examples θα βρείτε παραδείγματα της γλώσσας που θα υλοποιήσετε.
